width:너비, px/%/vw/vh/vmax/vmin/auto
height:높이, px/%/vw/vh/vmax/vmin/auto
※ 인라인 요소는 미디어만 제외하고 크기지정 불가

margin:바깥여백,
margin:값1 -> 상하좌우의 여백을 한번에
margin:값1 값2 -> 상/하 좌/우
margin:값1 값2 값3 -> 상 좌/우 하
margin:값1 값2 값3 값4 -> 상 우 하 좌
margin-top: 위 바깥여백만
margin-bottom: 아래 바깥여백만
margin-left: 좌 바깥여백만
margin-right: 우 바깥여백만
인라인 요소는 상/하 여백 적용되지 않음

padding: 안 여백,
padding:값1 -> 상하좌우의 여백을 한번에
padding:값1 값2 -> 상/하 좌/우
padding:값1 값2 값3 -> 상 좌/우 하
padding:값1 값2 값3 값4 -> 상 우 하 좌
padding-top: 위 안쪽여백만
padding-bottom: 아래 안쪽여백만
padding-left: 좌 안쪽여백만
padding-right: 우 안쪽여백만
인라인 요소는 상/하 여백 적용되지 않음

box-sizing:지정한 콘텐츠의 width와 height을 어디까지를 계산할지 방법 제공
content-box:기본값으로 width, height, padding, border, margin을 각 각 계산
border-box:지정한 width, height에 padding과 border를 포함하여 계산

가시 속성

overflow, overflow-x(가로), overflow-y(세로) : 흘러 넘치는 영역을 어떻게 처리할 지 설정 -> visible, auto, scroll, hidden

display:출력(화면에표시하는)방식 -> none, block, inline, inline-block, table

visibility:가시 -> visible, hidden

opacity:불투명도, 0(투명)~1(불투명) ex) opacity:0.5(반투명)
